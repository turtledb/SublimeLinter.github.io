    About SublimeLinter

    SublimeLinter is a framework for linting code in Sublime Text 3. The actual linting is done by separate Sublime Text 3 plugins, which can be installed via Package Control. SublimeLinter has been designed to provide maximum flexibility and usability for users and maximum simplicity for linter authors.
